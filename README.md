# Sorting Algorithms

## Description
This project is part of the Holberton School curriculum.  
It focuses on understanding and implementing different sorting algorithms,
their time complexities using Big O notation, and choosing the appropriate
algorithm depending on the input data.

## Learning Objectives
At the end of this project, I am able to explain:

- At least four different sorting algorithms
- What Big O notation is and how to evaluate time complexity
- How to choose the best sorting algorithm depending on the input
- What a stable sorting algorithm is

## Sorting Algorithms Covered
- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort

## Big O Notation
Big O notation describes the performance or complexity of an algorithm.
It focuses on how the execution time grows relative to the input size.

Examples:
- O(1): Constant time
- O(n): Linear time
- O(nÂ²): Quadratic time
- O(log n): Logarithmic time
- O(n log n): Linearithmic time
